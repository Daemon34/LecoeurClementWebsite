## How to launch the project locally ?

First, make sure you have npm installed. Then open a command prompt and type the following commands :
```bash
npm install
npm run dev
```
It should open your web browser, or you can access [http://localhost:3001/LecoeurClementWebsite/](http://localhost:3001/LecoeurClementWebsite/)

## How is the project deployed ?

The project has a GitHub Action configured to deploy the build on branch `gh-pages`. You can see all the steps of the GitHub Action on the [deploy.yaml](https://github.com/Daemon34/LecoeurClementWebsite/blob/main/.github/workflows/deploy.yaml]) file inside the .github/workflows folder
